# Playground one-seamless-scroll

用于本地演示 `@eosway/one-seamless-scroll`。

## 启动

```bash
pnpm install
pnpm --filter one-seamless-scroll dev
```

默认会启动一个最小可运行页面，用来展示：

- `OneSeamlessScroll` 组件真实渲染
- `list`、`minItems`、`speed`、`hoverPause`、`enabled` 的参数效果
- `state-change` 事件输出
- 空态、静态、滚动、悬停暂停等状态表现

## 构建

```bash
pnpm --filter one-seamless-scroll build
```
